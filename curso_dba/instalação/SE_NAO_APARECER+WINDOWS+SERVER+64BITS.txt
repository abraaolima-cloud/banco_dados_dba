Se não aparecer windows 2019 (64-bit), ou qualquer opção de servidor 64 bits, é porque provavelmente a BIOS do seu computador
veio de fábrica configurada para não rodar Máquinas virtuais, mas é fácil corrigir.

Pesquise no google:
"nao aparece windows 64 bits virtual box bios"

Vejo por exemplo este link:
